# Welcome to ZenGIS 🧘‍♂️

Transitioning from a general Computer Science or IT background into **Geographic Information Systems (GIS)** and **Remote Sensing** can feel overwhelming. There are new coordinate reference systems, map projection geometries, spatial databases, web map servers, and browser rendering engines to master.

**ZenGIS** is a modular, structured directory designed to make this onboarding process smooth and clear for junior developers.

---

## 🎯 What Can You Do with This Site?

This portal acts as a curated concept library for different engineering roles:

*   **🌐 Frontend Developers**: Learn how to build interactive 2D web maps (Leaflet, OpenLayers, MapLibre), render 3D globes (CesiumJS), and visualize massive datasets on the GPU (Deck.gl).
*   **💻 Backend Developers**: Master publishing maps via standard services (WMS, WFS), serving binary vector tiles (MVT), and scripting spatial APIs.
*   **🗄️ Database Developers**: Learn spatial SQL queries, joins, and optimization strategies using PostgreSQL and the **PostGIS** extension.
*   **⚙️ DevOps Engineers**: Containerize GIS applications, setup proxy tile caches (MapProxy, GeoWebCache), and tune database memory and map servers for production workloads.
*   **📊 GIS Analysts**: Master desktop mapping, layer symbology, geoprocessing tools inside **QGIS**, and automate workflows using Python (GeoPandas, Shapely).

---

## 🧭 How to Navigate the Portal

1.  **Start with the Foundations**: If you are brand new to GIS, explore the **General GIS** and **Remote Sensing** tabs in the top navigation bar. Make sure you understand coordinate reference systems (CRS), vector vs. raster data models, and how satellite imagery bands work before diving into code.
2.  **Explore by Topic**: Use the top horizontal tabs to navigate to your specific technical area (Frontend, Backend, Databases, DevOps, QGIS).
3.  **Search Instantly**: Looking for a specific function or library? Use the **Search bar** in the top header to find matching concept pages instantly.
4.  **Hands-on Exercises**: Each page contains a brief notes overview, link recommendations, and a **Practice Challenge** at the bottom to test your understanding.
